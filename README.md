# 🎓 ClubsConnect

**ClubsConnect** is a comprehensive college club management platform designed to streamline communication, event coordination, alumni mentorship, and collaboration across multiple colleges and clubs.

---

## 🚀 Features

### 👥 User Roles
- **Admin** – Manage colleges, faculties, and overall moderation
- **Faculty** – Oversee clubs, approve events, mentor students
- **Student** – Join clubs, participate in events, connect with alumni
- **Alumni** – Offer mentorship, post opportunities, engage with clubs

### 🏛️ Core Modules
- **Multi-College & Club Support**
- **Event Lifecycle Management** – Proposal → Approval → Promotion → Execution → Feedback
- **Mentorship System** – Connect students with alumni
- **Social Feed** – Posts, comments, likes, and flagging with AI moderation
- **Opportunity Portal** – Internships, jobs, scholarships shared by alumni
- **Chat System** – Person-to-person, group, and club discussions
